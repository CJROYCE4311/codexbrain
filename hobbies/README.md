# Hobbies

Add a subfolder per hobby. No fixed schema — define the structure that makes sense for each.

Example structure for a hobby:
```
hobbies/
└── woodworking/
    ├── projects.md       # Project log with status and notes
    ├── tools.csv         # Tool inventory
    └── resources.md      # Books, YouTube channels, suppliers
```

To add a new hobby category, say: "Add a hobby category for [name]"
